# Healthcare Claims Analysis



## Overview

This project analyzes healthcare claims to identify patterns in patient visits, diagnoses, provider activity, and claim amounts. The analysis helps highlight trends, high-value claims, and actionable insights for healthcare operations.



## Tools

- Excel (Raw Data, Pivot Tables, Charts)



## Excel Workbook

All data, analysis, and reports are included in a single Excel workbook:



- \*\*Raw Data\*\* – Original claims dataset including patient, provider, specialty, diagnosis, and claim amount.  

- \*\*Analysis\*\* – Pivot tables, aggregations, and visualizations to identify patterns by provider, specialty, age, and diagnosis.  

- \*\*Summary/Insights\*\* – Executive summary with key findings, charts, and actionable recommendations.



## Key Highlights

- Cardiology generates the highest total claim amounts.  

- Patients aged 50–60 contribute the most high-value claims.  

- Dermatology has frequent smaller claims that may benefit from efficiency improvements.  

- Neurology claims are high-value despite lower claim counts.  



## Objective

Demonstrate healthcare analytics skills by cleaning, summarizing, and visualizing claims data in Excel to provide actionable insights for operational decision-making.
