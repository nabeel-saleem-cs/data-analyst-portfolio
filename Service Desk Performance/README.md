# 📊 Service Desk Performance Optimization

## 🔍 Project Overview  
Analyzed service desk ticket data to evaluate SLA compliance, operational efficiency, and customer satisfaction. The goal of this project is to identify performance gaps and provide data-driven recommendations to improve service delivery.

---

## 📁 Dataset  
- Ticket categories (Network, Software, Hardware, Access)  
- Priority levels and SLA metrics  
- Resolution and escalation data  
- Customer Satisfaction (CSAT) scores  

---

## 📈 Analysis Performed  
- Ticket distribution by category and priority  
- SLA compliance assessment  
- Escalation vs. resolution trend analysis  
- Resolution time vs. CSAT correlation  

---

## 🛠️ Tools & Technologies  
- **Excel:** Data cleaning, aggregation (SUMIFS, COUNTIFS, AVERAGEIFS), lookups (VLOOKUP, XLOOKUP), pivot tables, charts  
- **Power BI:** Interactive dashboard and data visualization  

---

## 💡 Key Insights  
- Access tickets have the highest volume across all categories  
- SLA compliance is low, particularly for high-priority tickets  
- Escalations exceed resolutions, indicating workflow inefficiencies  
- Average resolution time (~24 hours) contributes to low CSAT (~2.98)  

---

## 🚀 Recommendations  
- Improve escalation and resolution workflows  
- Identify root causes of high escalation rates  
- Reduce resolution times through process optimization  
- Implement real-time SLA monitoring  

---

## 📊 Deliverables  
- Excel report summarizing performance insights  
- Power BI dashboard with interactive filters and drill-down capabilities
